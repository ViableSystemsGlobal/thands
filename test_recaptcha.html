Quick Setup Guide

  1. Go to Google reCAPTCHA Admin
  2. Create a new site with reCAPTCHA v2 "I'm not a robot" checkbox
  3. Add these domains:
    • localhost
    • 127.0.0.1
    • your-production-domain.com
  4. Copy the site key and secret key
  5. Add to your .env file: 
    VITE_RECAPTCHA_SITE_KEY=your-site-key-here
RECAPTCHA_SECRET_KEY=your-secret-key-here
  6. Replace "YOUR_SITE_KEY_HERE" in this file with your actual site key
  7. Restart your development server
